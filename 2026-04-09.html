Agent Orchestration New

x402 Protocol Joins Linux Foundation with Google, OpenAI, Circle

Google, OpenAI, and Circle have joined a Linux Foundation initiative backing the x402 protocol for autonomous agent payments. The protocol leverages the HTTP 402 "Payment Required" status code to enable instant, automated payments between agents. A competing standard — the Machine Payments Protocol (MPP) backed by Stripe and Paradigm — is also vying for dominance. The standards war signals that agentic commerce infrastructure is moving from experimental to production-ready.

Tax Optimisation New

Perplexity Launches Computer for Taxes

Perplexity announced Computer for Taxes, an AI agent that drafts full U.S. federal tax returns on official IRS forms. The agent reviews financial documents, asks follow-up questions, and produces completed returns — or audits a professionally prepared return for accuracy. Tax modules are continuously updated from IRS materials without model retraining. This represents the first consumer-facing agentic tax product from a major AI company, directly challenging TurboTax's $15B industry.
